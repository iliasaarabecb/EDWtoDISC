Helper functions and classes to get deal submissions metadata from EDW's database:

This module implements various auxiliary functions/classes which are used to keep track of the deal submissions (across different PCD's) within EDW's database.

Create a namespace object from a dict, recursively

Source code in edwtodisc\_tonamespace.py 
def __init__(self, adict):
    """Create a namespace object from a dict, recursively"""
    self.__dict__.update({k: self.__elt(v) for k, v in adict.items()})
